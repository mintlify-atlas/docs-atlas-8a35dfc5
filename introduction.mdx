## What is OpenCode Agents?

OpenCode Agents is a curated registry of **69 AI agent templates** for [OpenCode](https://opencode.ai), distributed via a zero-dependency CLI and interactive TUI. Each agent is a specialized system prompt that configures AI assistants for specific tasks — from TypeScript development to Kubernetes architecture, from security audits to product management.

Every agent follows a rigorous 4-section format: **Identity**, **Decisions**, **Examples**, and **Quality Gate** — ensuring high-quality, production-ready assistance.

## Quick Example

Install an agent in seconds:

<CodeGroup>

```bash npx (recommended)
npx github:dmicheneau/opencode-template-agent install typescript-pro
```

```bash Interactive TUI
npx github:dmicheneau/opencode-template-agent
# Auto-launches TUI — browse with arrow keys, select with Space, install with Enter
```

```bash Install a pack
npx github:dmicheneau/opencode-template-agent install --pack backend
# Installs: postgres-pro, redis-specialist, database-architect, api-architect, python-pro, typescript-pro, debugger, test-automator
```

</CodeGroup>

## Why OpenCode Agents?

Traditional AI assistants lack domain expertise. OpenCode Agents provides battle-tested system prompts that configure AI models with:

- **Proven decision trees** — IF/THEN rules for common scenarios
- **Real-world examples** — Code patterns from production systems
- **Quality gates** — Validation criteria before delivering work
- **Version pinning** — References to specific tool versions and best practices from 2024+

Instead of generic responses, you get expert-level guidance tailored to your exact use case.
